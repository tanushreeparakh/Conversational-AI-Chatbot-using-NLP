# Conversational AI Chatbot using NLP

## Project Overview

This project is a Conversational AI Chatbot developed using Python and Natural Language Processing (NLP) techniques. The chatbot understands user queries in natural language and provides relevant responses by matching them with a knowledge base using TF-IDF Vectorization and Cosine Similarity.

## Features

* Natural Language Query Processing
* Tokenization
* Stopword Removal
* Lemmatization
* TF-IDF Vectorization
* Cosine Similarity Matching
* Interactive Web Interface using Streamlit
* Knowledge Base of Technical Questions

## Technologies Used

* Python
* NLTK
* Scikit-learn
* Pandas
* Streamlit

## Installation

1. Clone or download the project.

2. Install dependencies:

pip install -r requirements.txt

3. Run the application:

streamlit run app.py

## NLP Techniques Used

### Tokenization

Breaks text into individual words.

### Stopword Removal

Removes common words such as "is", "the", "a".

### Lemmatization

Converts words into their root form.

### TF-IDF Vectorization

Transforms text into numerical vectors.

### Cosine Similarity

Finds the most relevant response by comparing user queries with stored questions.
